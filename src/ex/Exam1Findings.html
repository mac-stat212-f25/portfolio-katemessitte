Interactive Visualizations Exploration

Source 1

Chapter 13: Interactive Graphs (from Modern Data Visualization) https://rkabacoff.github.io/datavis/Interactive.html?utm.com Key findings / takeaways: - plotly is a powerful interface: you can wrap a ggplot object in ggplotly() to automatically get interactivity (tooltips, zoom, pan, legend toggling). - You can customize which fields show up in the tooltip by mapping extra variables to aesthetic labels. - The interactivity features (zoom, pan, reset, download) are built in by default.

Code
library(ggplot2)
library(plotly)

# Using the built-in mpg dataset
p <- ggplot(mpg, aes(x = displ, y = hwy, color = class)) +
  geom_point(size = 3) +
  labs(x = "Engine displacement (L)",
       y = "Highway mpg",
       color = "Car class") +
  theme_minimal()

# Turn the ggplot into an interactive plotly object
interactive_p <- ggplotly(p)

# Print / display it
interactive_p

## Interactive Visualizations Exploration

# Source 1
Chapter 13: Interactive Graphs (from Modern Data Visualization)
https://rkabacoff.github.io/datavis/Interactive.html?utm.com
Key findings / takeaways:
- plotly is a powerful interface: you can wrap a ggplot object in ggplotly() to automatically get interactivity (tooltips, zoom, pan, legend toggling).
- You can customize which fields show up in the tooltip by mapping extra variables to aesthetic labels.
- The interactivity features (zoom, pan, reset, download) are built in by default.
```{r}
library(ggplot2)
library(plotly)

# Using the built-in mpg dataset
p <- ggplot(mpg, aes(x = displ, y = hwy, color = class)) +
  geom_point(size = 3) +
  labs(x = "Engine displacement (L)",
       y = "Highway mpg",
       color = "Car class") +
  theme_minimal()

# Turn the ggplot into an interactive plotly object
interactive_p <- ggplotly(p)

# Print / display it
interactive_p
```

# Source 3
The Impact of Interactive Data Visualization on Decision-Making in Business Intelligence
https://www.researchgate.net/publication/381265855_The_Impact_of_Interactive_Data_Visualization_on_Decision-Making_in_Business_Intelligence
Key findings / takeaways:
- Using interactive visualizations improves data exploration, speeds decision-making, and enhances collaboration among stakeholders.
- However, the benefit depends on data quality, user skill level, and interface design. Poor design or messy data can negate the advantages.
- The authors caution against overly complex interactions, which can cause cognitive overload in users and reduce effectiveness. ResearchGate
